# SomethingDigital_BackToTop
Adds a "Back to Top" link that when clicked, sends the user to the top of the page.

Install via Modman
----------------

You can install this module using [Colin Mollenhour's](https://github.com/colinmollenhour) [Modman tool](https://github.com/colinmollenhour/modman).

```bash
$ modman init
$ modman clone https://github.com/sdinteractive/SomethingDigital_BackToTop
```
